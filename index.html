An ultra-efficient Node.js framework built on top of H3 - fully configurable and packed with a rich suite of out-of-the-box modules for developers who demand performance.

Core Features

Vercube comes with a powerful set of features designed to make server development faster and more enjoyable.

High Performance

Built on top of H3 engine for exceptional speed with minimal overhead.

Type Safety

100% TypeScript checked with OOP principles and decorators.

Best Developer Experience

Intuitive APIs and powerful tooling that makes development a joy.

Runtime Agnostic

Works seamlessly across Node.js, Deno, Bun and serverless environments.

Zero Configuration

Get started in seconds with zero configuration needed. Vercube works out of the box while remaining fully customizable.

That's it! A complete Vercube project requires zero configuration to run while still giving you the flexibility to customize every aspect when needed.

vercube.config.ts 
import { defineConfig } from '@vercube/core';

export default defineConfig({ });

Built-in Modules

Vercube comes with a comprehensive suite of modules that work together seamlessly, saving you time and effort.

Auth

Complete authentication solution with support for sessions, JWT, OAuth and more.

CLI

Powerful command-line tools for scaffolding, building, and managing your application.

Core

Essential foundations for your application with routing, middleware, and utilities.

Logger

Structured logging with multiple transport options and configurable log levels.

DevKit

Development tools to enhance productivity, debugging, and development workflows.

DI

Powerful dependency injection system for better code organization and testability.

Storage

Unified storage API with providers for Redis, Cloudflare KV, Amazon S3, and more.

Database

Database integration with providers for Drizzle ORM and Prisma.
